📌 ¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado que da estructura y significado al contenido web. No es un lenguaje de programación, sino de etiquetas que el navegador interpreta para mostrar textos, imágenes, enlaces, etc.

🏷️ Etiquetas básicas que aprendí

  • <a> – Hipervínculo (atributos: href, target).
  • <img> – Imagen (src, alt, width).
  • <div> – División genérica para agrupar.
  • <p> – Párrafo.
  • <ul> / <ol> / <li> – Listas desordenadas/ordenadas.
  • <span> – Contenedor de texto en línea.

📌 Ejemplo práctico (propio)

Para el laboratorio, hice una pequeña página de recetas:

<h1>Mi receta favorita</h1>
<img src="tacos.jpg" alt="Tacos" width="300">
<p>Ingredientes:</p>
<ul>
    <li>Tortillas</li>
    <li>Carne</li>
    <li>Cebolla</li>
</ul>
<a href="https://www.recetas.com" target="_blank">Más recetas</a>

Así comprendí cómo los atributos modifican el comportamiento (por ejemplo, target="_blank" abre en nueva pestaña).
