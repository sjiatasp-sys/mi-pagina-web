📌 ¿Qué es MySQL?

MySQL es un sistema de gestión de bases de datos relacional (RDBMS) de código abierto. Es súper rápido, escalable y usado por empresas como Facebook, Twitter y YouTube. Aprendimos a crear una instancia llamada UMG con usuario root y contraseña.

✨ Características principales

  • Código abierto – Gratuito y con gran comunidad.
  • Multiplataforma – Windows, Linux, macOS.
  • Seguridad – Control de usuarios y cifrado.
  • Rendimiento rápido – Ideal para aplicaciones web.

📌 Ejemplo práctico (propio)

En el laboratorio, usando MySQL Workbench, creé una base de datos para una tienda ficticia:

CREATE DATABASE tienda;
USE tienda;
CREATE TABLE productos (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nombre VARCHAR(100),
    precio DECIMAL(10,2)
);
INSERT INTO productos (nombre, precio) VALUES ('Laptop', 750.00), ('Mouse', 25.50);
SELECT * FROM productos;

Así entendí cómo se almacenan los datos en tablas con filas y columnas, y cómo las claves primarias evitan duplicados.
